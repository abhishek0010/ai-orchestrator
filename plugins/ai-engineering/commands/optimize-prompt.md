Rewrite and optimize an AI prompt for better performance and reliability.

Load the following expertise before starting:

- [prompt-engineering](../../../skills/prompt-engineering/SKILL.md)

## Process

1. Take an existing prompt or description of a task.
2. Apply systemic optimizations:
   - Restructure with clear headings and role definition.
   - Inject specific constraints and anti-patterns.
   - Add chain-of-thought instructions where appropriate.
   - Use delimiters (###, ---) to separate context from instructions.
   - Introduce 1-3 few-shot examples if applicable.
3. Version control the prompt if it's part of a plugin or agent instruction.
4. (Optional) Run a test against a representative model to verify the improvement.

## Rules

- Follow the [prompt-engineering](../../../skills/prompt-engineering/SKILL.md) checklist.
- Prioritize reliability and precision over "creative" flow.
- Use explicit XML-like tags or Markdown headings for better parsing.
